# iCal24
This application provides easy-to-use APIs proxied from Skola24.
The primary purpose of the application is to provide an iCalendar schedule feed extracted from Skola24.

If you do not wish to host the application yourself, you may use [ical24.pallhed.se](https://ical24.pallhed.se).

In order to use the application, you need to get your school/district's Skola24 hostname. You will also need a scope. ([ical24.pallhed.se](https://ical24.pallhed.se) is prefilled with the hostname and scope for schools in Örebro kommun.)

## Installation
1. Clone the repository.
2. Build & run the Dart project.
	- The easiest way to do this is by using [Docker](https://docker.com) containers. The project comes with a Dockerfile already setup.
	- You can also run the application natively with [Dart](https://dart.dev). See [dart: The Dart command-line tool](https://dart.dev/tools/dart-tool) for help on running or compiling the application.

## API
### Getting a list of schools
``/schools/<schoolHostname>/<schoolScope>``

Request URL: ``https://ical24.pallhed.se/schools/orebro.skola24.se/8a22163c-8662-4535-9050-bc5e1923df48``

Example response:
```json
[
	{
		"guid": "ZjBiZTJhODQtYWU3Mi1mYWI0LTg1NGMtYTdlMmQ5YzQzYjE1",
		"name": "Tullängsgymnasiet"
	},
	{
		"guid": "YjkzY2RkYjctNDg1NS1mNWE1LWIyNWItYWM0YjhmZGYyZjk2",
		"name": "Gumaeliusskolan"
	},
	...
]
```
### Getting a list of classes
``/classes/<schoolHostname>/<schoolGuid>/<schoolScope>``

Request URL: ``https://ical24.pallhed.se/classes/orebro.skola24.se/ZjBiZTJhODQtYWU3Mi1mYWI0LTg1NGMtYTdlMmQ5YzQzYjE1/8a22163c-8662-4535-9050-bc5e1923df48``

Example response:
```json
[
	{
		"guid": "ZTVjY2MwMTQtYTM4OS1mM2Y3LTk1YjItMTYyZDVjNzIxZGQ1",
		"name": "TEIM20"
	},
	{
		"guid": "Mzg4ZGY4YmYtYTc1ZS1mM2JkLTk4NDktMjZiYWNhNDg3NDQ5",
		"name": "TEIM21"
	},
	...
]
```

### Getting a schedule in [iCalendar](https://en.wikipedia.org/wiki/ICalendar) format
``/ical/<schoolHostname>/<schoolGuid>/<schoolScope>/<classGuid>?weeks=<int>``

The ``weeks`` query parameter defaults to 1. Can be any integer between 1 and 10.

Request URL: ``https://ical24.pallhed.se/ical/orebro.skola24.se/ZjBiZTJhODQtYWU3Mi1mYWI0LTg1NGMtYTdlMmQ5YzQzYjE1/8a22163c-8662-4535-9050-bc5e1923df48/Mzg4ZGY4YmYtYTc1ZS1mM2JkLTk4NDktMjZiYWNhNDg3NDQ5?weeks=3``
